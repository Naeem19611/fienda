# Findea - Premium Multi-Vendor E-commerce Frontend

Findea is a high-performance, modern e-commerce application built with **Next.js 15** and **TypeScript**. This project features a highly responsive design, modular component architecture, and a premium user interface tailored for a multi-vendor marketplace.

## 🚀 Project Overview
The primary goal of this phase was to develop the core user-facing pages with a focus on **responsiveness**, **performance**, and **aesthetic consistency**. Every page has been tested across mobile, tablet, and desktop viewports.

## 📁 Key Pages & Implementation Details

I have successfully designed and implemented the following pages within the `app/` directory:

### 1. User Authentication (Login)
- **Path:** `/login`
- **Features:** Optimized login form with validation-ready fields.
- **Design:** Clean and focused UI to improve conversion rates.

### 2. Account Dashboard
- **Path:** `/account`
- **Features:** A centralized hub for users to manage their profiles, settings, and personal data.
- **Responsiveness:** Grid-based layout that adapts seamlessly to smaller screens.

### 3. Order Management
- **Path:** `/order`
- **Features:** Clear list view of user purchases. Includes status badges and structured data for easy tracking.

### 4. Password Recovery
- **Path:** `/lost-password`
- **Features:** A user-friendly workflow for resetting forgotten passwords, ensuring security and ease of use.

## 🛠️ Technical Implementation (Tech Stack)

- **Framework:** Next.js 15 (App Router) for optimized routing and SEO.
- **Language:** TypeScript for type safety and better developer experience.
- **Styling:** Tailwind CSS for a utility-first, rapid UI development.
- **Icons:** `react-icons` library (used for consistent and lightweight iconography).
- **Typography:** **Playfair Display** for a premium, elegant look in headings and titles.
- **Components:** Custom modular components including Headers, Footers, and Mobile Drawers.

## 🎨 UI/UX Features
- **Responsive Navigation:** A custom mobile drawer for easy navigation on touch devices.
- **Modern Layouts:** Implementation of Glassmorphism and specialized trapezoid sections for a unique brand identity.
- **Single-Line Footer Policy:** Optimized footer copyright text that maintains a clean, single-line look across all devices.
